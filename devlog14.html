Devlog #14

Still Here

Posted November 11th, 2024

Hello World, Again

It has almost been a year since I've posted a devlog here. I started working full time, so that takes up the majority of my time now. Also, I haven't worked on this project consistently in quite a while either. But I have been chipping away at it.

What's New?

In short, I've completed/been working on the following features:

  • Fixed some minor & game breaking major bugs
  • Save player data during/after a run
  • Started saving players option choices (slider bar values, muted/unmuted)
  • Redid some visuals with a pixel art editor
  • New upgrade - passive income
New visuals:

Redoing some art has definitely reignited the fire to keep developing this game. I want to find an art style for this game, and then redo all buttons/sliders/enemies/etc.

What's Next?

I want to work/implement the following features:

  • Polish core game mechanics
  • Re-evaluate enemy spawning system / write a stronger wave spawning algorithm
  • Finalize art style
  • Add new player-based mechanics
  • Add reasons to keep playing the game, goals to work towards, in-game rewards

Theres Always Tomorrow

Though it has been a long time since I've worked on this project consistently, I keep realizing one thing. Although things might seem bleak in the moment, but as long as there is tomorrow, there is always the possibility that anything can happen. So while today I still have a lot I want to accomplish, I sit here knowing that it's all still possible, and will happen one day.
