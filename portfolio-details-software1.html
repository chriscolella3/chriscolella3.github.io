Software Engineering I

Software Engineering I consisted of learning the Software Development Lifecycle. We studied different types of life cycles like Sprint, Waterfall, Spiral, and Agile.

Our class simulated going through a Software Development Lifecycle, but instead of developing a real product, we focused on the documentation and the actual process itself.

This allowed me to get hands on experience as we were tasked with writing several documents. These documents being:

  • Software Requirement Specifications document (SRS)
  • Use and Test case documents
  • Functional requirements (FR) and non-FR documents
